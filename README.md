# Animated Dark Mode Toggle Button

## Overview

This project is a modern animated Dark Mode Toggle Button built using HTML and CSS. It features a smooth transition between light and dark themes, changing both the background and the toggle switch appearance when activated. The project demonstrates the use of advanced CSS selectors, pseudo-elements, animations, and transitions to create an interactive user interface without JavaScript.

## Features

* Smooth light mode and dark mode transitions
* Animated toggle switch movement
* Dynamic background color changes
* Pure HTML and CSS implementation
* Responsive and lightweight design
* Modern UI inspired by day/night theme toggles
* Uses CSS pseudo-elements for creating the toggle knob

## Tech Stack

* HTML5
* CSS3
